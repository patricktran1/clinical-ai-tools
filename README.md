# Clinical AI Tools

A collection of prototype tools designed to support clinicians through AI-driven analysis, triage support, and structured patient data extraction.

## Use Cases
- Dermatology referral triage (based on keywords and clinical urgency)
- Anomaly detection in intake data
- EMR summarization and decision routing

## Tech Stack
- Python, Jupyter
- OpenAI API (GPT-4-turbo)
- LangChain
- Pandas, scikit-learn, PyMuPDF

## Disclaimer
This repository is for educational and prototyping purposes only. Not intended for clinical deployment without validation and ethical review.
